- NTCIR12 dataset: 
http://www.cs.rit.edu/~rlaz/NTCIR12_MathIR_WikiCorpus_v2.1.0.tar.bz2

- Folder path contains dataset: 
NTCIR-12_MathIR_Wikipedia_Corpus\MathTagArticles\wpmath...

- The model runs on python 3.9, libraries used are in the requirements.txt file.

- The set of 20 query formulas provided by the NTCIR-12 competition is in the TestQueries folder, the set of 8 custom query formulas is in the CustomQueries folder.

- The results are saved in the Retrieval_Results folder.

- Tool to help evaluate retrieval results: Trec_eval tool (https://trec.nist.gov/trec_eval/)
